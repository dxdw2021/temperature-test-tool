# 温度测试工具

这是一个基于Python的温度测试工具，可以通过图像识别从万用表显示屏幕中读取温度数据，并进行记录和导出。

## 功能特点

- 实时温度显示
- 可自定义采样间隔和测试时长
- 数据实时记录和展示
- 测试完成后自动导出CSV文件
- 支持温度数值合理性检查

## 安装要求

1. Python 3.8+
2. Tesseract-OCR
3. Python依赖包

### 安装步骤

1. 安装Tesseract-OCR：
   - 访问 [Tesseract-OCR下载页面](https://github.com/UB-Mannheim/tesseract/wiki)
   - 下载并安装适合您系统的版本
   - 将Tesseract-OCR的安装目录添加到系统PATH环境变量中

2. 安装Python依赖包：
   ```bash
   pip install -r requirements.txt
   ```

## 使用说明

1. 运行程序：
   ```bash
   python temperature_test.py
   ```

2. 在界面中设置：
   - 采样间隔：每次读取温度的时间间隔（秒）
   - 测试时长：总测试持续时间（分钟）

3. 调整万用表位置：
   - 确保万用表显示屏幕位于屏幕指定区域内
   - 显示数字清晰可见

4. 开始测试：
   - 点击"开始测试"按钮开始记录
   - 可以随时点击"停止测试"按钮结束测试
   - 测试完成后数据会自动保存为CSV文件

## 注意事项

1. 首次运行前必须正确安装Tesseract-OCR并配置环境变量
2. 确保万用表显示屏幕在指定的截图区域内
3. 保持显示屏幕清晰，避免反光和遮挡
4. 如果识别失败，请检查：
   - Tesseract-OCR是否正确安装
   - 万用表位置是否正确
   - 显示是否清晰

## 故障排除

1. 如果提示"Tesseract-OCR未安装"：
   - 检查是否已安装Tesseract-OCR
   - 确认安装目录是否已添加到PATH中
   - 重启程序尝试

2. 如果无法识别温度：
   - 调整万用表位置
   - 确保显示清晰
   - 检查截图区域设置是否正确
