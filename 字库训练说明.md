# Tesseract 7段数码管字库训练指南

## 环境准备

1. **修复 TESSDATA_PREFIX**
   当前系统的 `TESSDATA_PREFIX` 指向了错误路径。打开 cmd 执行：
   ```
   setx TESSDATA_PREFIX "D:\Program Files\Tesseract-OCR\tessdata"
   ```
   或者直接在运行工具前先设置：
   ```
   set TESSDATA_PREFIX=D:\Program Files\Tesseract-OCR\tessdata
   ```

2. **确认训练工具**（已就绪）
   ```
   D:\Program Files\Tesseract-OCR\ 目录下已有：
   - text2image.exe
   - unicharset_extractor.exe
   - combine_tessdata.exe
   - lstmtraining.exe
   - combine_lang_model.exe
   ```

---

## 训练流程

### Step 1：收集训练图片

1. 把万用表切换到温度测量模式，显示不同的温度值
2. 用工具的 ADB 截图，通过"设置截图区域"框选出 LCD 数字区域
3. 保存 **5~10 张**不同数字组合的裁剪图（如 25.6、36.8、100.0 等）

### Step 2：准备训练文件

将裁剪好的图片放到 `training_images` 文件夹，重命名为：
```
seg7.lcd.exp0.tif
seg7.lcd.exp1.tif
seg7.lcd.exp2.tif
...
```
**注意**：必须是 `.tif` 格式。用画图或 Python 转换。
格式：`<语言>.<字体名>.exp<编号>.tif`

### Step 3：手动标注（用 jTessBoxEditor）

运行 `启动字库编辑器.bat` → 打开 jTessBoxEditor

1. 菜单栏 → **Box** → **Open** → 选择 `seg7.lcd.exp0.tif`
2. 会看到自动生成的框（不准确），需要手动调整
3. 每个框对应一个字符：
   - 拖动框的边缘调整大小
   - 在下方输入框输入正确的字符（0-9 或 .）
   - 用键盘左右键切换字符
4. 完成后 **Box** → **Save** → 保存为 `seg7.lcd.exp0.box`
5. 重复操作所有 exp 文件

### Step 4：生成 unicharset

```cmd
cd training_images
unicharset_extractor *.box
```

### Step 5：创建字体属性文件

创建 `font_properties` 文件（无扩展名），内容：
```
lcd 0 0 0 0 0
```

### Step 6：聚集训练数据

```cmd
cntraining .
mftraining -F font_properties -U unicharset -O seg7.unicharset .tr
```

### Step 7：生成 traineddata

```cmd
combine_tessdata seg7.
```

### Step 8：测试

将生成的 `seg7.traineddata` 复制到 `D:\Program Files\Tesseract-OCR\tessdata\`

在工具中设置 Tesseract 命令为：
```python
text = pytesseract.image_to_string(img, lang='seg7', config='--psm 7 --oem 1')
```

## 简化方案（推荐）

如果训练太复杂，可以先试试**讯飞 API**（已配置好密钥）：
1. 调整好 bbox 截取到 LCD 数字区域
2. 点击工具界面的 **"API识别"** 按钮
3. 讯飞云端 OCR 对 7 段数码管的识别率远高于 Tesseract

或者用 powershell 运行训练步骤：
```
powershell -ExecutionPolicy Bypass -File D:\DEV\tool\Test-Tool\train_font.ps1
```
